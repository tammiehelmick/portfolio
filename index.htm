Dictionary App

View the Dictionary App

The dictionary app is built in React Native. It makes use of the Merriam Websiter API to retrive data. A search feature allows a user to search for a word and recieve the definition as a response. The main screen also include a random word that is pulled from a list of interesting words. Finally there is a matching game to test a user's knowledge where they are presented with one definition and 4 words and they select the correc word.
